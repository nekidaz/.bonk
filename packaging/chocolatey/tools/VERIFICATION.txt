VERIFICATION
============
Verification is intended to assist the Chocolatey moderators and community in
verifying that this package's contents are trustworthy.

This package is maintained by the software author (nekidaz) and is published
automatically from the project's GitHub Actions release pipeline.

The installer is NOT embedded in this package. At install time it is downloaded
from the official GitHub release for the matching version:

  Project:  https://github.com/nekidaz/.bonk
  Releases: https://github.com/nekidaz/.bonk/releases

The exact download URL (`url64`) and its SHA-256 checksum (`checksum64`) are
embedded in `tools\chocolateyinstall.ps1`. Both are generated by the release
workflow directly from the GitHub release asset, so the checksum in the package
matches the published binary.

To verify manually:

  1. Open `tools\chocolateyinstall.ps1` and note the `$url64` and `$checksum64`.
  2. Download the installer from `$url64`.
  3. Run: Get-FileHash <installer> -Algorithm SHA256
  4. Confirm the hash equals `$checksum64`.
